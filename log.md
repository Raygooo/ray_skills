# Marketplace Evolution Log

Append-only record of how this marketplace grows itself — every skill/plugin distilled from a real workflow, every curation pass. Newest entries at the bottom. Borrowed from the `log.md` convention in Karpathy's LLM Wiki pattern.

Format (one line per event, parseable prefix):

```
## [YYYY-MM-DD] <kind> | <subject> | <note>
```

`<kind>` ∈ `distilled` (new skill/plugin), `updated` (revised), `curated` (lint/maintenance pass), `superseded` (replaced by something newer).
